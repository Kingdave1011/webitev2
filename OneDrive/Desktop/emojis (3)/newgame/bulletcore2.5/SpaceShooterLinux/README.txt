SPACE SHOOTER V2 - Linux Edition
==================================

HOW TO PLAY:
1. Extract all files from the ZIP
2. Open terminal in the extracted folder
3. Make the executable runnable:
   chmod +x "Space Shooter V2.x86_64"
4. Run the game:
   ./"Space Shooter V2.x86_64"

CONTROLS:
- Arrow Keys / WASD: Move ship
- Space / Click: Fire weapons
- ESC: Pause/Menu

FEATURES:
- Multiple ship types to choose from
- Wave-based enemy combat
- Progressive difficulty
- Settings customization
- Colorful menu buttons!

SYSTEM REQUIREMENTS:
- Linux (64-bit)
- 100 MB free space
- OpenGL 3.3 compatible graphics

TROUBLESHOOTING:
If the game doesn't start, make sure you've made it executable:
chmod +x "Space Shooter V2.x86_64"
